איך מתקינים את המפות?

  • בחרו את המפה הרצויה והורידו את הקובץ למכשירכם.
  • אם הקובץ ירד כארכיון (ZIP), יש לחלץ אותו כדי לקבל את קובץ ה-obf.
  • העתיקו את קובץ המפה לתיקייה נגישה בזיכרון הטלפון.
  • נווטו אל הקובץ באמצעות מנהל הקבצים שלכם, פתחו אותו ובחרו ב-OsmAnd כיישום הפתיחה.
